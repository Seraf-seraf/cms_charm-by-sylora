# Приемочный аудит готовности

Дата: 2026-07-15

## Вывод

Система **не готова к эксплуатации** до применения production-конфигурации и
подключения счетчика Метрики.
Незаполненные prod-данные товаров, категорий, контента, реквизитов и
юридических текстов в оценке не учитывались.

## Подтверждения заказчика

1. SMTP и production-доставка настроены.
2. Резервное копирование явно опущено заказчиком.
3. Вывод ошибок на production отключен.

## Осталось выполнить

1. Production-конфигурация веб-сервера не совпадает с зафиксированным шаблоном:
   HTTP редирект ведет на `https://charm-by-sylora.ru:443/`, `www` проходит два
   редиректа с промежуточным HTTP, `/config.php` возвращает HTTP 200 вместо
   запрета. Проверка 14.1 не закрыта.
2. Аналитика на production фактически не подключена: в HTML главной нет кода
   Яндекс Метрики или GA. Код модуля создан, но критерий 33 требует подключенную
   аналитику.
## Исправлено после аудита

- Добавлено руководство администратора по товарам и заказам.
- Loading-spinner отключается при `prefers-reduced-motion`.
- Корневые узкие PHP-тесты, audit/configure/smoke-test и их вызовы из CI/CD удалены по решению заказчика; CMS настраивается на production.

## Подтверждено

- `https://charm-by-sylora.ru/`, `/all-jewelry`, `/about`, `/contact` отвечают HTTP 200.
- Несуществующий URL отвечает HTTP 404.
- `/robots.txt` и `/sitemap.xml` отвечают HTTP 200.
- Canonical главной, каталога, страницы о мастере и контактов формируется.
- `https://pay.charm-by-sylora.ru/api/v1/health` отвечает HTTP 200 и `{"status":"ok"}`.
- PHP-синтаксис, PHPStan, production-assets и PhpCsFixer dry-run проходят.

## Ограничения аудита

- Не создавались заказы и платежи на production.
- Production-конфигурация и данные не изменялись.
- Оценка не учитывает незаполненные prod-данные, но учитывает обязательные
  эксплуатационные настройки и проверки из ТЗ.
