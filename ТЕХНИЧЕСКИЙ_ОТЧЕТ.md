# Технический отчет

Дата обновления: 2026-07-15

## Версии

- CMS: OpenCart `3.0.5.0`.
- PHP в текущем окружении: `8.5.8`.
- Composer-зависимости проекта: штатный набор OpenCart из `composer.json`, включая `twig/twig`, платежные SDK и `scssphp/scssphp`.

## Принципы реализации

- Базовая витрина дорабатывается на стандартной структуре OpenCart: PHP-контроллеры, модели, Twig-шаблоны, CSS и существующий JavaScript.
- Новые тяжелые frontend-фреймворки не добавлялись.
- Используются уже подключенные в OpenCart зависимости витрины: jQuery, Bootstrap, Font Awesome, Swiper.
- Основная визуальная система вынесена в `upload/catalog/view/theme/default/stylesheet/stylesheet.css` через CSS-переменные и компонентные классы.
- Управляемые данные берутся из OpenCart: категории, товары, фильтры, цены, скидки, наличие, изображения, инфостраницы, настройки магазина.

## Изменения системных файлов

Изменения ниже затрагивают системную часть OpenCart и должны учитываться при обновлениях CMS.

| Файл | Причина изменения |
| --- | --- |
| `upload/system/config/default.php` | Переключение дефолтной языковой директории и автозагрузки языка на `ru-ru`. |
| `upload/system/config/install.php` | Переключение языка установки/дефолтного языка на `ru-ru`. |
| `upload/system/library/language.php` | Fallback языка переведен на `ru-ru`, чтобы сайт не возвращался к старому языковому коду. |
| `upload/system/library/db/pdo.php` | Исправлена обработка `null` в `escape`, чтобы избежать ошибок на PHP 8.5. |

## Изменения общей логики витрины

- `upload/catalog/controller/common/header.php`: добавлена глобальная SEO-разметка и безопасная проверка HTTPS.
- `upload/catalog/controller/common/home.php`: главная выводит реальные категории и товары из моделей OpenCart.
- `upload/catalog/controller/product/category.php`: добавлены сортировки и фильтры каталога.
- `upload/catalog/model/catalog/product.php`: добавлены сортировка по популярности/новизне и фильтры по цене, наличию, новинкам, скидкам.
- `upload/catalog/controller/product/product.php`: подготовлены данные товара, атрибуты, блоки деталей и JSON-LD Product/Offer/BreadcrumbList.
- `upload/admin/view/javascript/summernote/opencart.js`: добавлено сопоставление языка `ru-ru` с локалью Summernote `ru-RU`.
- `upload/system/library/cdek_official`: добавлен адаптированный под PHP 8.5 модуль СДЭК с API v2, ПВЗ и управлением отправлением из заказа.
- `upload/system/library/russian_post_delivery.php`: добавлены упаковка корзины, fingerprint и серверная проверка тарифа Почты России.

## Масштабируемость по ТЗ

- Категории: структура берется из OpenCart, главная и каталог не завязаны на статичные категории.
- Фильтры: используются стандартные группы фильтров OpenCart для материала, цвета и типа украшения; дополнительно реализованы фильтры цены, наличия, новинок и скидок.
- Оплата: отдельная фискализация на сайте не закладывается, так как по ТЗ оплата должна работать через платежный агрегатор.
- Доставка: СДЭК рассчитывает варианты до двери и ПВЗ; Почта России использует ECOM-виджет для выбора отделения и серверный тарифный API. Оба способа включают стоимость в стандартные итоги OpenCart и сохраняют сведения в заказе.
- SEO: добавлены базовые meta/OG, canonical, JSON-LD Organization/WebSite/Product/Offer/BreadcrumbList и брендированная 404.
- Контентные страницы: страницы "Обо мне", "Контакты" и будущие доверительные страницы работают через стандартные маршруты OpenCart.
- Цветовая схема: основные цвета и темы управляются CSS-переменными.

## Ограничения текущего состояния

- Реальная структура категорий украшений еще должна быть настроена в админ-панели.
- Реальные товары, материалы, цвета, размеры, фото и alt-тексты зависят от финального ассортимента.
- SMTP и production-способы доставки настроены по подтверждению заказчика.
- Вывод технических ошибок на production отключен по подтверждению заказчика.
- Резервное копирование явно исключено заказчиком из текущей подготовки.
- Юридические тексты не являются юридически проверенными документами и должны быть предоставлены заказчиком или юристом.

- Системы вебмастеров: по подтверждению заказчика сайт добавлен в Яндекс.Вебмастер и Google Search Console.
